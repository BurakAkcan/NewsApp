# News App

In this project, I pulled data from News Api.


## Tech and Libraries

 - MVC Architecture
 - Load image from url with KingFisher
 - Network with UrlSession

  
## News API

100% Free & Open-source News API

- News Api Site (https://newsapi.org/).
